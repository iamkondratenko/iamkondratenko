Waypoint. life in motion

Waypoint
life in motion

Social network-tracker for active people

The Waypoint project is a concept of mobile application; it is like a mixed fitness tracker that can be used for sharing photos and locations when travelling. The Waypoint app has the potential to become a social network for people of new generation - active and healthy.

Why Waypoint is waypoint?

Way is a distance from the starting point to the ending point. Way contains the following data: distance, time spent on the road, burnt calories, method of travel (walking, running, riding a bike/skateboard, driving, etc.). Point is a certain place the user liked on his way. The user has the option of adding the Point while he is tracking his Way. The point contains the following data: latitude and longitude, location (park, forest, botanical garden, etc.)

Share activity

The user can track and share his activity. It could be a walk with a pet, a morning jog, cycling, travel to different cities and countries. Distance, time spent on the way, burnt calories - all this categories will be reflected on the user's profile.

scrollable screen

Discover new ways

The user's activities could inspire his subscribers to discover new routes for walks, open new places for hanging out, interesting destinations for trips with their families or friends. They will be surprised by how many interesting places are around them

scrollable screen

Trendy places

Trendy places are the most visited places or events around. Users can see popular streets, events and points in the city. It's a great tool for those who have no idea about where to go in a new city. Thus, if user is going to another city, he could plan the trip and easily find top places and events in the city.

scrollable screen

Event Points

The user can create an event, which he organises. For example, volunteers are planning a cleanup program for next weekend; all of the user's subscribers will receive notification about the event (location and time) on their devices.

Game Mode

The application's popularity depends of users involvement. Game mode - my best bet for growth of number of users. The concept of the application contains the necessary functionality for the game mode. For example: the moderator (user) create a game event - quest, and players (users) subscribe to it. The quest moderator in advance creates a route with Points. From the very beginning of the game players must search Points in augmented reality, using phone with the Waypoint app. If user finds Point then the next Point will be opened. The winner would be the user who first has opened all Points.

How it works

Take a moment and look at the application screens to understand how this app works

Home feed

Home feed shows all the Ways and Points, news and tags saved by the people which user choose to follow. Tags can be routes and points, localities, events and cities etc.

Please take a moment, and scroll right on the screen

Start way

This is the start of the trip. The starting location is determined by the current location via geolocation. After clicking Start Way your route starts to track. Please take a moment, and click on Start Way button.

Add the Point

A user can add Points, when he sees interesting places, or spectacular views while he is on his way.A user take a picture or record a video and create the Point with coordinate on the map. All user‚Äôs subscribers will see this point, after user publishes it.

Endpoint

A user can edit the Point (add photos or video), or add a new Point when he finished his way. On the right side of the screen there is a screen application "edit and share".

Please take a moment, and scroll right screen

Give a Highfive if you like it

A user can give a Highfive if he likes the Way or the Point. The Highfives mark works the same way as "like" in other social networks. Also, if a user gives the Highfive on another user's profile he automatically subscribes to this user.

Frame control

Frame control - it's a great function which shows moments in motion. A user has full control under video's timeline. If the user swipes his finger to the left - video starts, if he swipes to the right - video plays backwards. When user holds his finger on the screen he can slow down the video, or speed it up, or fully stop the video.

Augmented Reality

As mentioned before, AR is good for game mode. But we also could use AR for other purposes. Not all people like to use maps. However if given them opportunity to use augmented reality for searching destinations - it will be more simple and usable.

Profit

The Waypoint app could make a profit from usual promotion and media advertising. Moreover, the app will be receiving a lot of data about the user (interests, places user visited, subscriptions etc). This data could help advertiser to set target audience more accurately.

Thank you for reading :)

Mainbox concept app

Mainbox
concept app

Application for planning your day

"If you can't complete the task
less than a day - break it down!"

Create target

The main philosophy of this application sums up to: "If you can't complete the task less than a day - break!" This app has a greate tool for it - Targets.
I will explain you by example. I want to lose wieght. Maybe for some people it's not a problem, but not for me ;-)
So I created the Target of "Lose 25 kg." and I created a plan with everyday little tasks to reach the goal by the time I set.

Create tasks

I created the target. But this is not enough, because I need someone tell me - "Now you have to do morning exercises" or "Get up and go to the jog, else you always be fat!". Therefore in this app I created special tools for this purposes - three type of tasks. Why three? Okay, let's figure it out

Regular task

It‚Äôs a simple task which will remind you one time. For example, I want to lose 30 kg, but it's realy big step for me. Because I want to see the progress. So I create the task "Lose 10 kilos by December 31". The app will remind me to weigh myself.

Circle task

Which will remind me about my task every hour, day, week etc. Do I need to create it on and on? No, it's not usable. There is no need, as we have created a special tool for it - Circle task. Yes, you understand me right. It's task that will be repeated by the time. In my case, I created task "Jogging". Great! This task will be reminding me on the selected time

If-then task

If-then planing is realy interesting thing, also known as implementation intentions. It‚Äôs insanely simple yet undeniably one of the most effective ways to ensure goal success. If-then plans specify when, where and how you will take specific actions to reach your target. For example, if my task "Morning exercises" from target "Lose 25 kg" Is not completed, then app will recommend me the task "Jogging‚Äù. This simple actions help create or break habbits

Mainbox

This screen contains all targets and tasks that user created

Inbox

Unlike the previous screen this one contains tasks for a day sorted out by priority and time.
The task you need to perform today are displayed on one screen

Done

Who doesn't like to observe self achievements?
Everyone likes it, because it gives motivation and energy for new targets

Good luck with achieving your targets! :)

Elementum periodic table

Elementum
periodic table

The best interactive periodic table

Yeah Mr. White! Yeah Science!

I think an interactive periodic table is one of those things that I really wanted to do.
In my opinion, a modern periodic table should be beautiful and usable. So, I decided to do it. The best way to become acquainted with my Periodic table is to try it.
However, before that let me introduce what I've done

Element

The Periodic table has a legend about each of the elements, and can change the color of the legend if you need a filter by categories or blocks. In the future, I am going to expand the functionality. There will be a possibility to filter by atomic weight, atomic number, etc.

Category filter

As I‚Äôve said before, my periodic table has the possibility to filter by categories and blocks. In my opinion it's very useful. For example, if you need to select elements from the category "Transition metal‚Äù; you have to click on the filter and all categories besides your selected will become grayed out. It's really useful! Is it not?

Details

Now, after you‚Äôve selected the category, it's a good time to read more information about the elements. My periodic table has a pop up window for this, with all the common information in table for better perception.

No JS challange

One more thing! Maybe you've noticed, this site was made as an one-page application. But I decided to challenge myself, and I've made a periodic table without Java Script. Yes, all the transition effects, pop-ups and even filter logic, I've made using only HTML5/CSS3. It was realy useful experience
